系统架构总览

模块化单体架构 — 前后端分离

架构核心理念

单个 Spring Boot 应用，内部按 教务/行政/AI 三大模块清晰分层。前端 Vue3 SPA 按模块路由组织。后期可轻松拆分为微服务。

系统架构全景图
🌐 浏览器
Vue3 + Element Plus SPA
⬇ HTTPS 请求
🔐 Spring Security + SSO 认证
JWT Token / 角色权限拦截
⬇
📚 教务模块
Controller / Service / Mapper
🏛️ 行政模块
Controller / Service / Mapper
🤖 AI 助手模块
Controller / Service / LLM API
⬇
🗄️ 数据持久层
MyBatis-Plus + MySQL

技术栈明细

层级 技术选型 说明
前端框架 Vue 3 + TypeScript + Vite 响应式、组合式 API
UI 库 Element Plus 中文生态最完善
状态管理 Pinia Vue 3 官方推荐
路由 Vue Router 4 路由守卫 + 角色权限
后端框架 Spring Boot 3.x + Java 17 长期支持版本
ORM MyBatis-Plus 简化 CRUD
数据库 MySQL 8.0 关系型数据库
认证 Spring Security + JWT SSO 模拟 + JWT Token
API 文档 SpringDoc (OpenAPI 3) Swagger UI
AI 集成 OpenAI / 通义千问 API 可切换后端模型
